# Simple Calculator

This project contains a small Python calculator app with support for:

- add
- subtract
- multiply
- divide

## Requirements

- Python 3

## Run the app

From the project folder, run:

```bash
python git.py
```

## How to use

1. Choose one of these operations: `add`, `subtract`, `multiply`, or `divide`
2. Enter the first number
3. Enter the second number
4. View the result

## Notes

- Division by zero is handled with an error message.
- The app accepts whole numbers and decimals.
